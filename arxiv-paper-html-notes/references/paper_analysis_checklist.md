# Paper Analysis Checklist

Use this before final delivery.

## Source hierarchy

1. arXiv abstract page and PDF.
2. Official GitHub, Hugging Face, project page, institution page.
3. Paper supplementary material if available.
4. Search snippets only as discovery; verify claims in primary sources.

## Required checks

- Title matches arXiv.
- arXiv version is recorded.
- PDF URL is correct.
- Dataset numbers are copied from paper text, table, abstract, or official model card.
- Method description includes backbone, pretraining objective, adaptation, aggregation, and downstream heads when available.
- Validation description includes task count, cohorts, baselines, metrics, and clinical/prospective/reader evidence when available.
- Open-source status is conservative:
  - Use confirmed official links when available.
  - Otherwise write `未检索到明确公开代码/权重`.
- Figure captions identify paper figure source or state that the diagram is self-drawn.

## HTML checks

- All local images exist.
- All links are clickable.
- HTML parser reports no unclosed tags.
- Hero navigation anchors resolve.
- No left/right paper-body mismatch.
- No horizontal overflow at desktop and mobile widths.
- Bottom credit is present:
  - `AI-assisted paper analysis · Generated by Codex,`
  - `📝 Paper Analysis & Curation by CocoSun`

## Red flags

- Do not invent code/model release links.
- Do not quote large PDF passages.
- Do not use a paper figure without a caption that marks it as a paper figure.
- Do not leave placeholder text such as TODO, unknown, or lorem ipsum.
